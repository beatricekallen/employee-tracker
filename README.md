# Employee Tracker

This application uses MySQL, Node.js, and Inquirer to allow the user to
view and manage the departments, roles, and employees at their company. The user can:

- view all departments;
- view all roles;
- view all employees;
- add a department;
- add a role;
- add an employee; and
- update an employee role.
